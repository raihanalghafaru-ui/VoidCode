Belajar coding itu mudah
Apa itu Coding?
PENGERTIAN CODING
Coding adalah proses menulis instruksi dalam bahasa pemrograman agar komputer melakukan tugas tertentu. Komputer hanya mengerti bahasa mesin (biner 0 dan 1), namun coding memungkinkan manusia menulis menggunakan bahasa tingkat tinggi seperti Python, JavaScript, atau Java. Instruksi tersebut nantinya akan diterjemahkan oleh compiler atau interpreter menjadi perintah yang bisa dijalankan mesin. Coding adalah fondasi dari semua aplikasi, website, game, AI, hingga sistem operasi. Dengan belajar coding, kamu melatih logika, problem solving, dan kreativitas.

// Contoh kode sederhana

# Python

Cara Menjalankan Program
Run Kode & Alat yang Dibutuhkan
1. Compiler vs Interpreter: Compiler (C++, Go) menerjemahkan seluruh kode sekaligus menjadi file executable. Interpreter (Python, JavaScript) menjalankan kode baris per baris.
2. Cara Run Python: Install Python dari python.org, tulis kode di file dengan ekstensi .py, lalu buka terminal/CMD dan ketik `python namafile.py`. Bisa juga pakai IDE seperti VS Code, PyCharm, atau Pydroid3 di HP.
3. Cara Run JavaScript: Cukup buka browser, tekan F12 -> Console, atau buat file .html dan sisipkan script.
4. Cara Run HTML/CSS: Cukup double klik file .html di browser. Untuk live server bisa pakai extension Live Server di VS Code.
5. Cara Run Java: Install JDK, compile dengan `javac Main.java` lalu jalankan `java Main`.
6. Online Compiler: Untuk belajar cepat bisa gunakan replit.com, onecompiler.com, atau Google Colab (Python).

Bahasa Pemrograman Populer
Python - Otak AI & Data
Python adalah bahasa serbaguna dengan ekosistem kaya: Django (web), Pandas (data), TensorFlow (AI), dan otomatisasi. Sintaksnya mirip bahasa Inggris, cocok untuk pemula. Digunakan di Google, NASA, dan startup. Cara menjalankan: `python script.py`.
print("Belajar Python itu asyik!")
angka = [x*2 for x in range(5)]
print(angka) # [0,2,4,6,8]
JavaScript - Web & Interaktif
JavaScript berjalan di browser maupun server (Node.js). Digunakan untuk membuat website dinamis, animasi, dan aplikasi realtime. Framework seperti React, Vue, Angular sangat populer. Bisa dijalankan di console browser atau file .js dengan node.
alert("Halo dari JS!");
document.querySelector("h1").style.color = "#facc15";
console.log("JS Modern");
Java - Enterprise & Android
Java dikenal dengan "Write Once Run Anywhere" karena JVM. Banyak dipakai di backend, aplikasi Android (Kotlin juga), dan sistem besar. Compile ke bytecode lalu jalankan di JVM.
public class Main {
public static void main(String[] args) {
System.out.println("Java rocks!");
}
}
Kotlin - Modern untuk Android
Kotlin adalah bahasa pemrograman modern yang berjalan di atas JVM (Java Virtual Machine). Dikembangkan oleh JetBrains, Kotlin menjadi bahasa resmi untuk pengembangan Android (selain Java). Sintaksnya lebih ringkas dan aman (null safety). Kotlin juga bisa digunakan untuk backend (Ktor, Spring), multiplatform mobile, dan JavaScript. Cara menjalankan: install IntelliJ IDEA, buat file .kt, compile dengan `kotlinc hello.kt -include-runtime -d hello.jar` lalu `java -jar hello.jar`. Bisa juga menggunakan playground online.

XML - eXtensible Markup Language
XML adalah bahasa markup yang dirancang untuk menyimpan dan mengangkut data. Mirip HTML, tapi lebih fleksibel karena tag-nya dapat didefinisikan sendiri. XML sering digunakan untuk konfigurasi aplikasi, pertukaran data antar sistem (SOAP, REST), file layout Android, dan RSS feed. Tidak memiliki perilaku bawaan seperti HTML, fokus pada struktur data. Cara menjalankan: cukup buka file .xml di browser atau editor teks, parser XML dapat membaca menggunakan JavaScript (DOMParser) atau berbagai bahasa pemrograman.

MySQL - Database Populer
MySQL adalah sistem manajemen database relasional (RDBMS) yang menggunakan SQL. Digunakan oleh banyak aplikasi web seperti WordPress, Laravel, dan e-commerce. Query dasar: SELECT, INSERT, UPDATE, DELETE. Sangat penting untuk backend developer.
CREATE TABLE users (
id INT AUTO_INCREMENT PRIMARY KEY,
nama VARCHAR(100),
email VARCHAR(100)
);
SELECT * FROM users WHERE nama LIKE 'Raihan%';
C++ - Performa Tinggi
C++ adalah bahasa pemrograman yang sangat cepat, digunakan untuk game engine, sistem operasi, embedded system, dan aplikasi real-time. Dikembangkan dari C dengan tambahan fitur OOP. Compiler seperti g++ digunakan untuk mengcompile kode C++.
#include <iostream>
using namespace std;
int main() {
cout << "Hello C++" << endl;
return 0;
}

Detail Pemrograman Web
HTML, CSS, JavaScript
HTML membangun struktur, CSS mempercantik tampilan, JavaScript menambahkan interaksi. Untuk menjalankan, buat file index.html, tulis kode dan buka di browser. Kamu juga bisa menggunakan Live Server untuk melihat perubahan secara real-time. Tips: Gunakan browser DevTools (F12) untuk debugging. Belajar Flexbox dan Grid membuat layout responsif. Media queries untuk tampilan mobile. JavaScript modern menggunakan ES6 seperti let, const, arrow function, dan fetch API.

Database & Backend

Database menyimpan data terstruktur. SQL digunakan untuk mengambil, menyimpan, memanipulasi data. Cara run SQL bisa pakai MySQL Workbench, phpMyAdmin, atau SQLite di terminal. Untuk backend, kamu bisa menggunakan Python Flask, Node.js Express, atau PHP. Contoh menghubungkan Python ke MySQL: install mysql-connector-python, lalu buat koneksi.
